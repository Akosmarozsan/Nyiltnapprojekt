Rendszer- és alkalmazásüzemeltető

A rendszer- és alkalmazásüzemeltető szakma fontos szerepet játszik a modern informatikai környezetben, mivel a vállalatok és szervezetek megbízható IT infrastruktúrára és folyamatosan működő alkalmazásokra támaszkodnak. Az ezen a területen dolgozó szakemberek felelősek a számítógépes rendszerek, hálózatok, valamint a különféle alkalmazások telepítéséért, konfigurálásáért, karbantartásáért és hibaelhárításáért.

Főbb feladatai:

  • Rendszerek telepítése és konfigurálása
  • Hálózati infrastruktúra üzemeltetése
  • Alkalmazások karbantartása
  • Hibaelhárítás és problémamegoldás
  • Biztonsági mentések és helyreállítások
  • Rendszerbiztonság fenntartása

A Petőfiben ezzel foglalkozunk:

  • Windows operációs rendszer
  • Linux operációs rendszer
  • Virtualizáció
  • DHCP
  • IPv4 és IPv6
  • Cisco Packet Tracer
  • HTTP és HTTPS
  • FTP

Szoftverfejlesztő

A szoftverfejlesztő és -tesztelő szakma a modern technológiai világ egyik legfontosabb területét foglalja magában. A szoftverfejlesztők és tesztelők felelnek a számítógépes programok, mobilalkalmazások és egyéb digitális megoldások létrehozásáért, valamint azok minőségének biztosításáért.

Főbb feladatai:

  • Fejlesztés
  • Hibajelentések készítése
  • Hibajavítás
  • Automatizált és manuális tesztelés
  • Rendszertervezés
  • Dokumentáció készítése

A Petőfiben ezzel foglalkozunk:

  • HTML
  • CSS
  • Bootstrap 5
  • Python
  • JavaScript
  • C#
  • Wordpress
  • PHP
